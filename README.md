<div align="center">
  <h1>custom-cli</h1>
  <p>Support multi pages and single page.</p>
</div>
<h2 align="center">Usage</h2>

``` bash
# install global
yarn global add custom-cli
```

``` bash
# create project
custom create projectname

choose template: vue or react
```

```bash
# install dependencies
cd projectname && yarn install
```
